# KLAX Signal Room UI Add-on

This add-on contains the Codex implementation prompt for the approved Prototype A dashboard plus the approved HTML and screenshot references.

Use it beside the authoritative current five-model package. It does not replace or modify the strategy package.

## Start

1. Copy this directory into the repository, for example:

   ```text
   implementation/klax_signal_room_ui/
   ```

2. Give Codex the contents of `CODEX_UI_IMPLEMENTATION_PROMPT.md`.
3. Keep the approved reference files available at the paths named in the prompt.

The requested implementation is read-only and must leave live, canary, taker, and order-submission paths disabled.
